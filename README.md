# 📎 프로젝트 소개

Hacker News Api를 활용한 news 모아보기 프로젝트입니다.

# 🛠️ 기술스택

React.js, Sass, HTML, CSS

# 💾 설치 및 실행방법

```
git clone https://github.com/BobaeKeum/HackerNews.git

npm install
//for yarn User : yarn install

npm start
//for yarn User : yarn start
```
